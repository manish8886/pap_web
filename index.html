RTJ-PAPRAZZI

A Real Time Java Implementation of Paparazzi UAV.

Real Time Java is emerging as a new technology for developing embedded and real time applications. As a result of this, interest in Real Time Java is increasing rapidly amongst researchers as well as in industry for exploring the challenges in its usage and its impact on development of real time systems. The goal of this project is to provide researchers as well as technology enthusiast to experiment with the features of Real Time Java. In order to accomplish this goal we aim to develop an open source implementation of the autopilot core of the PAPARAZZI UAV in Real Time JAVA powered with Fiji VM and RTEMS as an underlying Real Time Operating System.
